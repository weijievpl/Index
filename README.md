# AR 粒子手势控制

基于 MediaPipe 和 Canvas 的手势交互粒子系统，可通过手势缩放、拨动粒子效果。

## 🎮 在线体验
[点击访问](https://你的用户名.github.io/仓库名)

## ✨ 功能特性
- 🤖 MediaPipe AI 手势识别
- ✨ 实时粒子特效
- 🖐️ 五指张开/握拳控制缩放
- 👆 食指拨动粒子交互
- 📱 响应式设计，支持移动端
- 🔄 前后摄像头切换
- 🎨 多种图案模式（文字、心形、自定义图片）

## 🚀 快速开始
1. 访问项目链接
2. 点击"🚀 启动"按钮
3. 允许摄像头权限
4. 在摄像头前展示手势：
   - 🖐️ 张开手掌：放大粒子
   - ✊ 握拳：缩小粒子
   - 👆 食指：拨动粒子
   - 🔄 切换镜头：前后摄像头切换

## 🛠️ 技术栈
- HTML5 Canvas
- MediaPipe Hands API
- Vanilla JavaScript (ES6+)
- CSS3 Flexbox

## 📱 兼容性
- Chrome/Edge 最新版 ✅
- Safari 14+ ✅
- Android Chrome ✅
- iOS Safari ✅
- 要求 HTTPS 环境

## 🚨 注意事项
- 首次加载需要下载 AI 模型（约20MB）
- 需要在光线充足的环境中使用
- 建议保持手势距离摄像头30-50cm
